Wiki를 사용하면 GitHub의 다른 곳과 마찬가지로 콘텐츠를 작성할 수 있습니다. 자세한 내용은 GitHub에서 작성 및 포맷 시작을(를) 참조하세요. 오픈 소스 Markup 라이브러리를 사용하여 다양한 형식을 HTML로 변환하므로 Markdown 또는 지원되는 다른 형식으로 작성하도록 선택할 수 있습니다. Markdown을 사용하여 렌더링된 수학 식, 다이어그램, 맵 및 3D 모델을 wiki에 추가할 수 있습니다. 렌더링된 수학 식을 만드는 방법에 대한 자세한 내용은 "수학 식 작성"을(를) 참조하세요. 다이어그램, 지도, 3D 모델을 만드는 방법에 대한 자세한 내용은 "다이어그램 만들기"을(를) 참조하세요. 퍼블릭 리포지토리에서 wiki를 작성하는 경우 wiki는 일반 대중이 사용할 수 있습니다. 프라이빗 리포지토리에서 wiki를 작성한 경우 해당 리포지토리에 대한 액세스 권한이 있는 사람만 wiki에 액세스할 수 있습니다. 자세한 내용은 리포지토리 표시 유형 설정을(를) 참조하세요. GitHub에서 직접 wiki를 편집하거나 wiki 파일을 로컬로 편집할 수 있습니다. GitHub.com의 모든 사용자가 퍼블릭 리포지토리의 wiki에 기여하도록 허용할 수 있긴 하지만 기본적으로 리포지토리에 대한 쓰기 액세스 권한이 있는 사용자만 wiki를 변경할 수 있습니다. 자세한 내용은 Wiki에 대한 액세스 권한 변경을(를) 참조하세요.
